# Phase 5: Specialization Tracks

Duration: 4 weeks. Goal: job-ready expertise in chosen domain.

## Tracks

| Track | Focus | Career Path |
|-------|-------|-------------|
| A | DevOps/Cloud | Platform Engineer, SRE |
| B | Security Engineering | Security Analyst, SOC Engineer |
| C | Systems Programming | Systems Developer, Kernel Engineer |
| D | Embedded Linux | IoT Developer, Hardware Integrator |

Complete one track for the Linux Specialist Certificate. Complete all four for the Linux Grandmaster Certificate.

## Track A: DevOps/Cloud

Skills: Kubernetes, Helm, Terraform, GitOps, service mesh, cloud-native architecture.

Project: Deploy microservices application to Kubernetes with auto-scaling, GitOps, and observability.

## Track B: Security Engineering

Skills: Advanced hardening, IDS/IPS, forensics, penetration testing basics, threat hunting.

Project: Build SOC monitoring dashboard with IDS, FIM, and incident response playbooks.

## Track C: Systems Programming

Skills: C for systems, system calls, debugging, eBPF, kernel concepts.

Project: Build system monitoring tool in C using /proc, /sys, and eBPF.

## Track D: Embedded Linux

Skills: Cross-compilation, Buildroot/Yocto, hardware interaction, real-time systems.

Project: Build custom Linux distribution for Raspberry Pi with hardware control and OTA updates.
