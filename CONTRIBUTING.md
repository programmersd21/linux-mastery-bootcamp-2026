# Contributing

## Reporting Issues

- Check if the issue already exists
- Provide clear steps to reproduce
- Include OS and version
- Suggest a fix if possible

## Adding Labs

1. Follow the existing lab template format
2. Include: Objective, Starting State, Steps, Verification, Failure Cases
3. Test on a clean VM
4. Include a Challenge section

## Improving Documentation

- Fix typos and grammar
- Clarify confusing explanations
- Add analogies and visual aids
- Update outdated information

## Adding Cheat Sheets

- Keep to 80 characters width
- Include top 3-5 flags per command
- Add one-line examples
- Include common gotchas
