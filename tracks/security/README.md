# Track B: Security Engineering

Duration: 4 weeks. Certificate: Security Engineering Specialist.

## Skill Map

Linux Hardening -> Network Security -> Auditing -> Offensive Defense
- SELinux/AppArmor, kernel security
- Firewall/IDS, VPN, segmentation
- AIDE/OSSEC, log analysis
- Penetration testing, forensics

## Weekly Breakdown

### Week 21: Advanced Hardening

Build server passing CIS Level 2 with custom policies.

Topics: CIS Level 2 benchmarks, custom SELinux policies, kernel security parameters, AIDE, security auditing automation.

### Week 22: Network Security

Build secure network with IDS/IPS and VPN.

Topics: Suricata rule development, network segmentation, WireGuard/OpenVPN, Zero Trust basics, network forensics.

### Week 23: Digital Forensics

Build forensics toolkit for incident response.

Topics: Timeline analysis, file recovery (photorec, testdisk), memory forensics (Volatility), log correlation, chain of custody.

### Week 24: Penetration Testing Basics

Build penetration testing report for test environment.

Topics: Reconnaissance, scanning, vulnerability exploitation, privilege escalation, reporting, ethical hacking principles.

## Final Project: SOC Dashboard

Requirements:
1. CIS Level 2 compliant servers
2. Suricata with custom rules
3. AIDE with alerting
4. Centralized logging with anomaly detection
5. Automated vulnerability scanning
6. Incident response playbooks
7. Real-time security metrics dashboard
8. Security policies and procedures
